# Linktree Frontend

This is a Vite + React frontend scaffold for the provided backend.

Features:
- React + Vite
- Tailwind CSS
- Framer Motion
- Axios API client

Quick start:

1. Install dependencies

```bash
cd frontend
npm install
```

2. Start development server

```bash
npm run dev
```

The frontend expects the backend at `VITE_API_URL` (default `http://localhost:5000`). Edit `.env` if needed.
